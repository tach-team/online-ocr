## ADR Format

Each ADR follows this structure:

- **Title** - Short descriptive name
- **Status** - Accepted / Superseded / Deprecated
- **Context** - What problem or requirement prompted this decision
- **Decision** - What was decided
- **Consequences** - Resulting benefits and drawbacks

## Creating New ADRs

1. Copy template from existing ADR
2. Use next sequential number (e.g., `004-your-decision.md`)
3. Fill in all sections
4. Add to index table above
